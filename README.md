### Setup

1. **Clone the repository**
   \`\`\`bash
   git clone <your-repo-url>
   cd nepal-mart-app
   \`\`\`

2. **Setup Backend**
   \`\`\`bash

   npm run seed
   npm run dev
   \`\`\`

3. **Setup Frontend**
   \`\`\`bash
   cd frontend
   npm install

4. **Access the Application**
